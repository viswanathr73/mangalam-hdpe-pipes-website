# Mangalam HDPE Pipes -- Product Landing Page

A fully responsive product landing page built using **Vanilla HTML, CSS,
and JavaScript**. The project replicates a **Figma design** and follows
a **component-based CSS architecture** with section-level styling and
responsive layouts.

The page showcases **HDPE Pipes & Coils for modern infrastructure**,
including product information, technical specifications, applications,
manufacturing process, testimonials, downloads, and contact
interactions.

------------------------------------------------------------------------

# Project Overview

This project implements a **high-performance responsive landing page**
without using frameworks like React, Next.js, Bootstrap, or Tailwind.

All UI components are built using:

-   Semantic HTML
-   Modular CSS
-   Vanilla JavaScript

The layout strictly follows **Figma design specifications**, including
typography, spacing, colors, and responsive behavior.
